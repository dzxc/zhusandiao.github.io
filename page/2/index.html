nginx笔记

发表于 2016-05-10   |  

nginx

高性能的HTTP和反向代理服务器

特点

  1. 配置非常简单（nginx.conf）
  2. 高并发，非阻塞（最高支持50000个并发）
  3. 负载均衡

安装

brew install nginx

查看版本

nginx -v ：查看版本
nginx -V ：查看版本和配置信息

运行

nginx -s reload #重新加载配置
nginx -s reopen #重启
nginx -s stop #停止
nginx -s quit #退出
