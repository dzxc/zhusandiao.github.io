《语法俱乐部》GitBook 版的成书过程

发表于 2016-02-25   |   阅读次数

最近打算将旋元佑老师的那本口碑特别好的语法书–《文法俱乐部》，又称《语法俱乐部》电子化，这本书网上有三个版本，怎么会有三个版本呢？可以先来看看知乎上有关这个问题的解答。

具体的写作平台，我选择的是Gitbook，其实李笑来老师与别人合作做了一个本土化的Gitbook–知笔墨，但是现在已经暂时无法使用了，等知笔墨能够使用了，再迁移上去。

《语法俱乐部》的成书

参考的版本

我参考的版本是Kindle114 论坛 的二十二章节版本。想下载的自己点进去看看，或者找相关的知乎答案–里面提供了三个版本的下载。

客户端的 bug

为了编辑的便利，我是下载了 GitBook 的最新的 桌面客户端5.0beta版
遇到了一些问题，比如编辑比较卡顿，编辑过程会冲突，后来遇到了类似于图上的这个问题导致彻底无法同步，经过搜索确定是客户端 bug，Github 的相关 issue 在这里，索性就只使用网页版进行编辑。

使用到的插件

共使用了5个插件，如图：

插件名 作用介绍
betterchinese 优化中文显示
emphasize 给相关文字加背景色
hide-published-with 隐藏目录页下面的那行 published using GitBook
ungrey 当章节页本身没有内容时，不让它显示灰色不可点击的状态，直接显示下一页的内容，这个插件，实际没有使用，因为我想到了更好的处理办法。
word-count 不用说，计数的
几个 CSS 文件

GitBook 可以针对网页版，epub,mobi,pdf 不同的版本自定义 CSS ，我统一使用了如下的CSS： 

strong {
    color: brown;
}  
/* 使着重标记的文字显示成棕色 */ 

em {
    color: brown;
}  
/* 使文字倾斜并且显示成棕色 */

书籍 bug 的提交

发现书籍的问题，请直接发邮件到：zhusandiao@gmail.com
最好是截图并带有相关描述，以便我更快修复

书籍自定义封面

我自己制作的封面如博文开头所见，原图大小为 1.13M ，下载地址：Cover.jpg

制作过程呢，主要利用了这个字体云网站做了字体云的效果，然后再用图片编辑软件加上书籍标题和作者，就OK了。

纯净版 epub 格式下载

GitBook 上虽然提供了下载，但是速度很慢，这里提供一个去除了最后打赏章节的 epub 格式的下载：语法俱乐部.epub

最后，求打赏

做这本书，我自认为是很用心的，看起来好像不难，但是为了优化内容的显示，还是花了相当的心思的（如果直接复制原电子书的内容，整个文字会混作一团，非常难以辨别，需要进行文字的着重，变色和加下划线），整个制作过程用了超过100个小时，希望大家可以打赏杯咖啡钱，3、5块不嫌少，当然多多益善。可以扫码，或者直接转到：
