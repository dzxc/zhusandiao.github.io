Drawable Resources

A drawable resource is a general concept for a graphic that can be drawn to the screen and which you can retrieve with APIs such as getDrawable(int) or apply to another XML resource with attributes such as android:drawable and android:icon. There are several different types of drawables:
drawable resource 概念上来说就是以一种可以被绘制到屏幕上的图形，可以通过如 getDrawable(int)这样的API获取或者是通过如 android:drawable 和 android:icon 这样的属性应用到另一个 XML 资源上。有以下几种不同类型的 drawables:

  1. Bitmap File: 一个位图格式的文件，生成BitmapDrawable
  2. Nina-atch File: 一种带有课伸缩区域的PNG格式的图片，可以基于内容改变图片尺寸，后缀名是.9.png，生成NinePatchDrawable
  3. Layer List: 可以管理一个其它Drawable组成数组的Drawable，这些 Drawable 资源按照数组顺序绘制，所以索引值最大的在顶部绘制，生成LayerDrawable
  4. State List: 一种XML文件，用来在不同状态指向不同的位图图形（例如，当 button 被按下的时候，使用不同的图片），生成StateListDrawable
