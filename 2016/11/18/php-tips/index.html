代码注释可能出现的问题

单行注释：

仅仅注释到行末或者当前的 PHP 代码块，之后会跳出 PHP 模式，这可能会造成多输出了一些 HTML 代码

多行注释；

C 风格的多行注释在碰到第一个 */ 时结束。

简单来说，形如/* xxxx */ xxxxx*/，会造成注释与自己的预期不符，部分内容未被正确注释（可以通过编辑器的代码高亮来避免）
